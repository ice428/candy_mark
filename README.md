# CandyMark
CandyMark is a simple markdown editor.

![](./screen.png)

## Features
- Live preview
- GitHub Flavored Markdown support
- mermaid.js support
- Katex support
- PDF export support
- Fast Synchronized Scroll

## Usage
- `mermaid` code block will render mermaid diagrams.
- `math` code block will render KATeX.
